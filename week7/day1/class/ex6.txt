1. Основной класс Animal:

Создается класс Animal с конструктором, который принимает параметры name, type, и color и инициализирует соответствующие свойства объекта.
Добавлен метод makeSound, который принимает параметр sound и возвращает строку, описывающую звук животного.
2. Расширенный класс Mammal:

Создается класс Mammal, который расширяет (extends) класс Animal.
В конструкторе класса Mammal вызывается конструктор родительского класса Animal с использованием super для установки свойств name, type, и color.
Добавлен метод sound, который вызывает метод makeSound родительского класса Animal с использованием super.
3. Создание экземпляров:

Созданы экземпляры класса Mammal - myDog, myCat, и myDolphin, каждый со своими уникальными параметрами.
4. Вывод в консоль:

Выведены результаты вызовов метода sound для каждого экземпляра, передавая соответствующие звуки животных.
5. Результат:

Каждый вызов sound выводит строку, описывающую звук и детали каждого животного, используя иерархию классов и методы родительского класса через super.
